Q7.How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?

Ans.

Step 1: Install Node.js installer for windows. Click on this link. Here install the LTS version.

Step 2: Open command prompt to check whether it is completely installed or not type the command

node -v

Step 3: Now in the terminal run the below command

npm install -g create-react-app

Step 4:Create a new folder react app using the below command

newfolder name

Step 5: Now inside this folder run the command

create-react-app reactfirst YOUR_APP_NAME

step 6:and last step

npm start

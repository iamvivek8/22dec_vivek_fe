Q1.Explain Life cycle in Class Component and functional component with Hooks

Ans.

3 phases in the React Component LifeCycle

  • mounting phase
  • updating phase
  • unmouting phase
  • mounting

    • During the mounting phase, a functional component is being created and added to the DOM

  • updating

    • updating phase, the functional component is re-rendered due to changes in its props or state

  • unmouting

    • unmounting phase, the functional component is being removed from the DOM.
